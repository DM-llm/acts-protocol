## 🐛 常见问题

### 问题 1：端口被占用

**现象**：启动服务时提示端口已被占用

**解决方案**：
```bash
# 查看端口占用
lsof -i :3000

# 杀死进程
kill -9 {PID}
```

### 问题 2：数据库连接失败

**现象**：无法连接到数据库

**解决方案**：
1. 检查数据库服务是否启动
2. 检查 `.env` 中的数据库配置
3. 检查网络连接

### 问题 3：依赖安装失败

**现象**：依赖安装失败

**解决方案**：
```bash

{包管理器清除缓存命令}

# 重新安装
{重新安装命令}
```

### 前端日志

- **查看**：`tail -f logs/app.log`

### AI 服务日志
- **位置**：`logs/ai.log`
