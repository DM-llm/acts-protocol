## 📝 使用说明

### 更新频率
- 开始新任务时：添加到"当前进行中"
- 完成任务时：移到"最近完成"，创建 work-log
- 暂停任务时：移到"暂停的任务"

### 保持简洁
- 只保留 1-3 个当前任务
- 最近完成只保留最多 3 个
- 详细内容用指针指向具体文档

### 文件引用格式
```markdown
#[[file:context/docs/{模块}/{功能}/01-{文档}.md]]
```
